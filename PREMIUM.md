# Premium Roadmap (ERP Serralheria)

## Princípios
- Premium = módulo/feature atrás de Feature Flag + Role
- Sempre medir valor: reduz tempo / evita erro / aumenta margem / compliance

## Premium (alto valor)
1) Auditoria completa (UI audit trail, diff, export CSV)
2) NF-e / XML entrada (importar compra -> RECEIVED + custos + ledger)
3) Integrações de pagamento (Asaas/MercadoPago/Gerencianet) + conciliação
4) Workflows de aprovação (Compras: solicitar -> aprovar -> enviar -> receber)
5) Relatórios avançados (margem por produto via BOM, ABC, giro, produtividade)
6) PWA chão de fábrica (offline, QR na OP, fotos, checklist)
7) RBAC granular por ação + logs de segurança
8) Alertas inteligentes (estoque mínimo, atraso OP, inadimplência AR)

## Upsell rápido
- Multiempresa real (troca de empresa + tema + permissões)
- Exportações (Excel/PDF) e integrações (webhooks)
- Automação de preços: custo BOM -> preço sugerido (markup/margem)
